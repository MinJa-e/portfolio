상세 설명
코로나 회복의 기대감과 더불어, 다양한 기능을 담아볼 수 있는 주제인 여행 플랫폼을 기획.
주요 기능
  1. 스프링 부트 / myBatis / REST API CRUD 등을 이용한 각종 백엔드 기능 설계
  2. 스프링 시큐리티를 이용한 보안화 및 SNS 로그인 기능
  3. AJAX Fetch를 이용한 비동기 통신 - 유효성 검사 및 게시판류 등에 활용
  4. 슬릭 라이브러리를 이용한 메인 페이지 상품 추천 및 배너 삽입
  5. API를 이용하여, 결제 시스템 및 챗봇 시스템 구현
  6. 시리얼 넘버로 등록 가능하며, 유효 기간이 있는 쿠폰 기능 삽입
  7. 쿠폰 기능과 상품 정보가 들어간 예약 페이지
  8. 숙소 목록 필터링 및 정렬 기능
  9. 내 예약 및 결제 내역 확인
  10. 상세 페이지 내 리뷰 및 Q&A
  11. 포인트에 따른 등급 기능 삽입
  12. 위시 리스트 기능
담당 업무
  1. 프론트 팀장 및 백엔드 보조
  2. 프로젝트 기획 및 ERD 설계
  3. 와이어프레임을 이용한 대부분의 페이지 화면 설계
  4. HTML / CSS / JS / 부트스트랩 등을 이용하여 모든 페이지 구현
  5. 슬릭 / 채널톡 등 외부 API 구현
  6. 공지사항 / 문의 게시판 등 게시판류 백엔드 설계
  7. AJAX Fetch를 통한 비동기 통신 구현
사용 기술
  • Front-end : HTML / CSS / JS / 제이쿼리 / 부트스트랩 / 타임리프
  • Back-end : 스프링부트 / myBatis
  • Others : mySQL / MariaDB / H2 / IntelliJ / Notion / Gather
