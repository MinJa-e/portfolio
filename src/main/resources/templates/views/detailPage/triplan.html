주제
포트폴리오형 블로그
상세 설명
팀 프로젝트 때 백엔드 경험에 부족함을 느껴, 추가적인 공부 및 포트폴리오 제작의 필요성을 느껴 제작
단순한 포트폴리오 제작만 구현하기에는 아쉬워, 개인 창작물 전시 및 방명록 기능 추가
주요 기능
  1. 스프링 부트와 RESTful API를 이용해 로그인, 회원가입, 방명록, 페이지네이션 등 기본적인 웹 구조 구현
  2. 방명록 내 비밀글/답글 기능 구현
  3. 자바 스크립트와 부트스트랩을 이용하여 스크롤 리모컨, 모달, 콜랩스, 드롭다운, 아코디언, 경고 메세지, 토글, 그림자 등 동적 요소 구현
  4. 부트스트랩과 미디어 쿼리를 이용한 반응형 웹 페이지 구현
  5. 스프링 부트와 자바 스크립트를 이용한 유효성 검사
  6. 스프링 시큐리티를 이용한 데이터 암호화 구현
담당 업무
  1. 스프링 부트와 RESTful API를 이용해 로그인, 회원가입, 방명록, 페이지네이션 등 기본적인 웹 구조 구현
  2. 방명록 내 비밀글/답글 기능 구현
