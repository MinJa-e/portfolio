주제
아이스크림 후원 페이지
상세 설명
'Buy me a coffee'와 '도네리카노'라는 후원 플랫폼에 영감을 받아 제작.
결제 API와 JPA를 응용해보기 위한 목적으로 기획했으며, 밈으로 이용되는 이미지를 삽입해 친근함을 주었음.
주요 기능
  1. 후원 내용 작성 및 확인 기능을 위한 JPA 활용
  2. 아임포트의 API를 이용해 결제 기능 도입
사용 기술
  • Front-end : HTML / CSS / JS / 부트스트랩
  • Back-end : 스프링부트 / JPA
  • Others : MariaDB / IntelliJ / VSC

링크
이용 중인 호스팅 서비스의 JVM 메모리 부족으로 에러가 발생해, 해당 프로젝트의 호스팅은 잠시 보류중
느낀 점
  1. 관리자 입장에서 결제가 이루어지는 과정을 볼 수 있어, 결제 서비스에 대한 이해도가 향상 되었음.
  2. JPA를 프로젝트에 처음 실사용 해보면서, 객체지향적인 ORM의 특징에 대해 알게 되었음.
