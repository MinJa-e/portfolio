가위바위보 초기 페이지
가위바위보 로그 및 재도전 메세지
주제
Vue로 제작한 가위바위보 게임 페이지
상세 설명
프론트 엔드 프레임 워크에 대한 호기심 및 공부의 필요성을 느끼고, Vue를 통해 간단하게 만들어본 가위바위보 게임 페이지
주요 기능
  1. Vue의 데이터 바인딩 문법을 통해, 가위바위보 행동 구현
  2. Vue의 v-for 문법을 통해, 목숨 및 로그 출력 구현
  3. Vue의 클래스 바인딩 문법을 통해, 가위바위보 결과에 따른 로그 CSS 변화 구현
사용 기술
  • Front-end : HTML / CSS / JS / Vue
  • Others : VSC
