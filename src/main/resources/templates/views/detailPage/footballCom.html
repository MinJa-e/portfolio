글 쓰기 페이지
주제
축구 커뮤니티 제작
상세 설명
평소 즐겨보던 인스타그램 축구 페이지의 관리자분과 연이 닿아 제작한 축구 커뮤니티.
이미지 위주의 메인 게시판과 함께 프로토 타입을 제작하였으나, 기획 단계에서 진행 잠정 중단.
주요 기능
  1. 최신 게시글의 핵심 이미지들을 선택적으로 메인 페이지에 썸네일로 올릴 수 있도록 로직 구현
  2. 이미지를 많이 다루는 게시판 특성 상, 다양한 형식으로 글을 쓸 수 있게 'CK에디터'라는 위지윅 에디터 구현
  3. 스프링 내부에 구현 되어있는 인터페이스를 통해, 파일&이미지 업로드 구현
  4. MyBatis의 조건절과 타임리프의 'classappend'구문으로 검색 및 카테고리 탭 구현
  5. SQL에 있는 ORDER BY / LIMIT 등의 구문을 이용해 게시글의 이전 글과 다음 글을 읽어내는 로직 구현
  6. SQL의 UPDATE 구문을 이용해 게시글 조회수를 구현, 스프링 시큐리티 구현 후, 추천 및 인기글도 추가 예정
사용 기술
  • Front-end : HTML / CSS / Javascript / JQuery / Bootstrap / Thymeleaf
  • Back-end : Spring Boot / MyBatis
  • Others : MySQL / MariaDB / IntelliJ / VSC

느낀 점
  1. 보편적인 형태가 아닌 게시판( = 사용 이미지를 선택적으로 메인 페이지에 노출)을 주문 받아 구현이 어려웠었는데, 로직을 스스로 구현하면서 사용 언어들의 응용력이 향상 되었음.
  2. 카테고리/검색과 같은 기능으로 게시판을 읽어오는 로직의 다양화가 필요 했는데, 이 과정 속에서 MyBatis 및 SQL에 대한 이해도가 향상 되었음
  3. 기능 구현 중, 반복되는 코드가 자주 등장 했는데 배열이나 반복문 등을 이용한 코드 효율화의 중요성을 꺠닫게 됨
