상세 설명
팀 프로젝트 '트립플랜'을 진행 후에 백엔드 경험에 부족함을 느끼고, 추가적인 공부 및 포트폴리오 작성을 위해 제작
단순한 포트폴리오 제작만 구현하기에는 아쉬워, 개인 창작물 전시 및 방명록 기능 추가
주요 기능
  1. 스프링 부트 / myBatis / REST API CRUD / Ajax Fetch 등을 이용해 로그인, 회원가입, 방명록, 페이지네이션 등 기본적인 웹 구조 구현
  2. 자바 스크립트 / 부트스트랩 / 슬릭 등을 이용하여 스크롤 리모컨, 슬라이더 등 동적 요소 구현
  3. 부트스트랩과 미디어 쿼리를 이용한 반응형 웹 페이지 구현
  4. 스프링 부트와 자바 스크립트를 이용한 유효성 검사
  5. 방명록 내 비밀글/답글 기능 구현
  6. 스프링 시큐리티를 이용한 데이터 암호화 구현(예정)
사용 기술
  • Front-end : HTML / CSS / JS / 제이쿼리 / 부트스트랩 / 타임리프
  • Back-end : 스프링부트 / myBatis
  • Others : mySQL / IntelliJ / VSC
